# The Lean Language Reference - Index

Source: https://lean-lang.org/doc/reference/latest/

## About

"This is the _Lean Language Reference_. It is intended to be a comprehensive, precise description
of Lean: a reference work in which Lean users can look up detailed information, rather than a
tutorial intended for new users."

Covers Lean version 4.29.0-rc6.

Lean is "an **interactive theorem prover** based on dependent type theory, designed for use both
in cutting-edge mathematics and in software verification."

Key characteristics:
- Core type theory implemented in a minimal kernel
- Advanced automation through an expressive tactic language
- User-extensible features
- Pure functional programming language capabilities
- Primary implementation in Lean itself
- Each tactic produces a term in the core type theory checked by the kernel
- Run-time system based on reference counting
